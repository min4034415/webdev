Vite + Vue 프로젝트에 Tailwind CSS 설치하기

1. 프로젝트 생성

Vite를 사용하여 새로운 Vue 프로젝트를 생성합니다. Vite는 빠르고 효율적인 개발 환경을 제공합니다.

npm create vite@latest my-vue-app -- --template vue

이 명령어로 my-vue-app이라는 이름의 Vue 프로젝트가 생성됩니다.

2. Tailwind CSS 설치

Tailwind CSS와 Vite 플러그인을 설치합니다.

npm install tailwindcss @tailwindcss/vite

설치가 완료되면, Tailwind CSS를 프로젝트에서 사용할 수 있게 됩니다.

3. Tailwind 설정 파일 생성

Tailwind 설정 파일을 생성합니다. 이 파일을 통해 프로젝트에서 Tailwind가 어떤 파일들에 스타일을 적용할지 정의할 수 있습니다.

npx tailwindcss init

이 명령어로 기본 tailwind.config.js 파일이 생성됩니다. tailwind.config.js 파일을 열고, content 항목에 Tailwind가 적용될 파일들의 경로를 지정합니다.

/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html", // HTML 파일 경로
    "./src/**/*.{vue,js,ts,jsx,tsx}", // Vue, JS, TS 파일 경로
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

4. CSS 파일에 Tailwind 적용

프로젝트의 src 폴더 내에 assets/styles.css와 같은 CSS 파일을 만들어서 Tailwind 지시어를 포함시킵니다.

/* src/assets/styles.css */
@import "tailwindcss";

이 파일을 main.js 또는 main.ts에서 import하여 스타일을 적용합니다.

// src/main.js
import './assets/styles.css'

5. 프로젝트 실행

이제 모든 설정을 마쳤다면, 프로젝트를 실행하여 Tailwind CSS가 제대로 적용되는지 확인합니다.

npm run dev

문제 해결

만약 Tailwind CSS가 적용되지 않는다면, tailwind.config.js 파일에서 content 경로 설정을 다시 확인하세요. 또한, CSS 파일에서 @import가 제대로 작성되었는지 확인해야 합니다.

결론

위의 방법을 통해 Vite + Vue 프로젝트에서 Tailwind CSS를 설정하고, 간단한 스타일링을 적용할 수 있습니다. Tailwind의 유틸리티 클래스들을 활용하여 빠르게 디자인을 구현할 수 있습니다.
